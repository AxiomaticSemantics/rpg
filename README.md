# A prototype hack & slash RPG

An experimental RPG prototype built with [Bevy Engine][bevy].

_This project is in heavy development and targets a fork of Bevy with cherry-picked syncs of Bevy's main branch._
 
# What is working?
* splash screen
* background audio
* character creation
* a basic game loop
* a basic HUD
